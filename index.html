What CRUMB Actually Is

CRUMB isn't a productâ€”it's a foundation. Think Arduino, but for building AI companions. Once you have the physical device, you control what it becomes: a meditation timer, a focus tool, a gentle reminder system, an offline AI assistant, whatever your heart desires

Build Your Own

The hardware is simple by design. You'll need:

  • CRUMB Genesis (Start Here): ESP32 board + USB-C cable + heartbeat code
  • Cost: ~$12
  • Build time: 5 minutes (plug in, upload, done)
  • Skills needed: None
  • Then Expand: Add LEDs, buttons, sensors, displaysâ€”whatever you need
  • CRUMB for your desk? Sure. CRUMB you can talk to? Why not. CRUMB toothbrush? You bet.
  • The platform grows with youâ€”start simple, build complexity as you learn
  • You don't buy CRUMB, you build it. You don't accept what it does, you define it.

Everything's Open

Github repository includes everything you need to build, modify, and improve CRUMB GENESIS:

  • Complete hardware schematics, Lego like build guide
  • Code (Code needs to be added to the storage, you can either use mine or create your own!)
  • 3D printable cases (STL files)
  • Bill of materials with supplier links
  • Assembly guide for beginners

The point: You own it. You fix it. You build on it. The standard is the giftâ€”what you create with it is yours.
